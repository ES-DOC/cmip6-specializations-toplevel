CMIP6 - Model Specializations

Introduction The scientific model domain descriptions are defined for CMIP6 via "specializations" which define the properties needed to have a configured model.  They are organised following the eight official CMIP6 realms (see PCMDI github repository) – aerosols, atmosphere, atmospheric chemistry, land ice, land surface, ocean, ocean biogeochemistry, sea-ice  – plus one top level specialization.  These specializations constrain the basic CIM2 Model documents for these particular realms (by adding or removing properties as needed). These specializations are defined and governed by ES-DOC realm experts who interact with the scientific community, and include several levels of reviews. This approach clearly separates the concepts encapsulated in the CIM from the specific CMIP6 activity. Tooling Chain The specialization files are written in python which act as inputs to a set of code generators that emit artefacts such as:
  • Mind-Maps (for visualization purposes);
  • JSON files (for configuration purposes);
  • IPython notebooks ((for documentation creation purposes);
  • Online questionnaire (for documentation creation purposes).
Review Process The review process for each realm is as follows:
  • Start from CMIP5 version and update with a few experts (version 0.X);
  • Organise review by community at large and include feedbacks;
  • Official release of version 1.0 for CMIP6 operational use (1st June 2017);
  • Correct any major error by 15th Sept 2017 (version 1.1);
  • Freeze for rest of CMIP6.
